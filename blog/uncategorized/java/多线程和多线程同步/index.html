二、多线程同步

2.1 什么是多线程同步

同一个进程内的多个线程会共享数据，当对共享数据进行并发访问时会出现Race Condition 即竞争状况。

多线程同步是指：

  • 协调多个数据对共享数据的访问，避免出现数据不一致的情况。
  • 协调各个事件的发生顺序，使得多线程在某个点交汇并按照预期步骤往前推进，比 如某线程需要等另一个线程完成某项工作才能开展该线程的下一步工作。

2.2 为什么需要同步
