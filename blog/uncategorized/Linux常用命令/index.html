文件上传

scp（Secure Copy）是 Linux/Unix 系统中常用的远程文件复制命令，它基于 SSH 协议进行数据传输，加密、安全，可以在本地与远程主机之间，或两台远程主机之间复制文件/目录。以下是完整用法与示例。

1️⃣ 基本语法

  • 源路径/目标路径可以是：

    • 本地文件/目录：直接写本地路径

    • 远程文件/目录：user@host:/path/to/file

      • user：远程用户名

      • host：远程主机 IP 或域名

      • /path/to/file：远程文件路径

2️⃣ 常用参数

参数 作用
-r 递归复制整个目录（复制文件夹时必带）
-P 指定 SSH 端口（注意是大写 P）
-p 保留原文件的修改时间、访问时间、权限
-v 显示详细调试信息
-C 传输时启用压缩，提高大文件传输效率

找到对应的进程 并杀死

  • Linux:
    • sudo lsof -i :8080
    • sudo netstat -tulpn | grep 8080
    • sudo kill -9 8080
  • Window:
    • netstat -ano | findstr 8080
      • taskkill /PID 8080 /F
